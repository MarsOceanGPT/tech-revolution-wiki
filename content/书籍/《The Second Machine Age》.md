# 《The Second Machine Age》

> 一句话：数字技术正在对认知劳动做蒸汽机对体力劳动做过的事——而且速度更快。

## 核心内容

Brynjolfsson和McAfee将当代数字化视为一个新的工业革命。三个特征定义了第二次机器时代：指数增长（摩尔定律的持续作用使计算能力以指数速度增长）、数字化（信息的零边际成本复制改变了经济学基础）、以及组合式创新（当指数增长的技术与重新组织的流程结合时爆发）。

核心论点：我们正处于Paul David描述的生产率悖论的又一轮——技术的能力已经爆发式增长，但组织结构、教育体系、法律制度等社会制度仍在按线性速度适应，导致在GDP统计数据上看不到生产率增长。这个滞后期可能持续数十年。

## 对本项目的价值

将Paul David的历史性生产率悖论分析（针对电力）延伸到当代AI和数字化时代。维度E（技能断裂）的核心理论参考。如果第一次机器时代花费了40年才在统计数据中显现生产率收益，那么第二次机器时代可能也需要类似的时间让教育、制度和组织形态适应。这解释了为什么AI在技术上已经突破，但社会还处于"无马马车"阶段。
